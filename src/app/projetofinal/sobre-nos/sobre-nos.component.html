Sobre o Projeto

Uma plataforma escolar de jogos desenvolvida com paixão e dedicação

O Projeto

O Vortex Games é um projeto escolar desenvolvido como parte do curso, com o objetivo de criar uma plataforma interativa onde estudantes possam descobrir, explorar e compartilhar seus jogos favoritos. Este projeto combina design moderno, funcionalidade intuitiva e uma experiência de usuário envolvente.

A plataforma foi desenvolvida utilizando tecnologias web modernas como Angular, TypeScript, PrimeNG, HTML e CSS demonstrando conhecimentos práticos em desenvolvimento front-end.

Ano do Projeto
